## The Problem

AI coding assistants struggle when your documentation is scattered, outdated, or formatted for humans instead of LLMs. They hallucinate conventions, reinvent patterns you've already established, and drift from your architectural decisions.

**Cortex TMS solves this.** It scaffolds a project structure that AI agents can parse, navigate, and trustâ€”keeping your codebase and its documentation in perfect sync.

<GlassQuote author="Cortex TMS Team" liquidBlob={true}>
  We dogfood everything. This project uses Cortex TMS to build itselfâ€”our NEXT-TASKS.md tracks development, our PATTERNS.md guides AI collaboration, and our Guardian CLI reviews every commit.
</GlassQuote>

## Quick Installation

```bash
# Install globally
npm install -g cortex-tms@2.7.0

# Or use with npx (no installation required)
npx cortex-tms@2.7.0 init
```
