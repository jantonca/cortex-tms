## The Challenge We're Solving

Modern AI tools like Claude Code are incredibly powerful, but they come with a hidden cost: **context explosion**. Without Cortex TMS, your AI assistant reads:

- Old documentation nobody maintains
- Deprecated code paths
- Irrelevant test fixtures
- Configuration for platforms you don't use

Result? **Higher costs, slower responses, worse suggestions.**

<Terminal language="typescript">
// Without Cortex TMS
const context = await readEverything() // 66,834 tokens
const cost = calculateCost(context)    // $0.20 per query

// With Cortex TMS tiered memory
const context = await readHotTier()    // 3,647 tokens
const cost = calculateCost(context)    // $0.01 per query
const savings = "94.5% reduction"      // ‚ú®
</Terminal>

<Callout type="tip" title="Token Savings Explained">
Cortex TMS uses a **tiered memory system** (HOT/WARM/COLD) to give AI only what it needs:

- **HOT** (Active): Current work context - AI reads this
- **WARM** (Truth): Documentation, patterns - On-demand only
- **COLD** (Archive): Old code, deprecated files - Ignored

Smart context = better AI suggestions at 10x lower cost.
</Callout>
